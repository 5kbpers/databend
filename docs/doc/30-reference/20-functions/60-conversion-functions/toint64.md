---
title: toInt64
---

toInt64 function is used to convert a specified value to a 64-bit signed integer.

## Syntax

```sql
toInt64(expr) — Results in the Int64 data type.
```

## Examples

```sql
mysql> select toInt64(123);
+--------------+
| toInt64(123) |
+--------------+
|          123 |
+--------------+
```
