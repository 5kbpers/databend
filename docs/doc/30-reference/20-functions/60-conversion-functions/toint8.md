---
title: toInt8
---

toInt8 function is used to convert a specified value to a 8-bit signed integer.

## Syntax

```sql
toInt8(expr) — Results in the Int8 data type.
```

## Examples

```sql
mysql> select toInt8(123);
+-------------+
| toInt8(123) |
+-------------+
|         123 |
+-------------+
```
