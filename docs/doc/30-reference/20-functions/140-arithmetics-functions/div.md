---
title: div
---

Returns the integer quotient of the division of num1 by num2. 

## Examples

```sql
MySQL [(none)]> select 5 div 2;
+-----------+
| (5 div 2) |
+-----------+
|         2 |
+-----------+

MySQL [(none)]> select div(5, 2);
+-----------+
| div(5, 2) |
+-----------+
|         2 |
+-----------+
```
